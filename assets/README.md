# Assets

Struttura consigliata assets del template.

- icons: icone statiche app e feature
- images: immagini raster png/jpg/webp
- illustrations: assets grafici illustrativi
- animations: lottie/rive o animazioni custom
- fonts: font custom progetto
- audio: suoni, effetti, musica
- json: file statici di configurazione/mock

Regole:

- file naming in snake_case
- evita file non usati nel template finale
- comprimi immagini prima del commit
