# ASO e store kit

## Contenuti da mantenere aggiornati

- descrizione breve
- descrizione lunga
- keyword set per lingua
- screenshot e mockup
- video preview (se presente)
- privacy policy

## Struttura cartella store

- store/description.txt
- store/keywords.txt
- store/privacy_policy_template.md
- store/screenshots/

## Processo consigliato

- definisci valore unico app
- allinea keyword con feature reali
- aggiorna screenshot a ogni release importante
- traccia conversione da listing a install
