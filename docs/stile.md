# Design system

## Regola fondamentale

Nessun colore, radius o spacing hardcoded dentro le feature.

## Primitive

- AppColors
- AppTypography
- AppSpacing
- AppRadius

## Componenti condivisi

- AppBottomSheet
- AppToast
- SmartBanner
- RewardedButton

## Coerenza UI

- usa semantica colori (primary/success/warning/error)
- centralizza text style per gerarchie tipografiche
- usa spacing costante (4/8/12/16/24/32)
- usa radius predefiniti per cards e bottoni

## Smooth libraries

- onboarding con smooth_onboarding
- paywall con smooth_paywall
- bottom sheet con smooth_bottom_sheet
- feedback con smooth_toast

## Regole UX

- evita schermate monolitiche
- componi per sezioni riutilizzabili
- usa bottom sheet per azioni contestuali
